# track-expenses
💰 個人記帳管理系統  
🌟 核心功能  
🔐 帳號管理  
• 安全登入：支援個人帳號登入系統。  
• 密碼修改：使用者可隨時更新帳號密碼以確保安全。  
• 帳號註銷：提供刪除帳號功能，清除所有相關個人資料。  


📝 帳務操作  
• 視覺化主頁：主畫面採用左右雙表格設計，分別顯示 「收入」 與 「支出」，上方顯示總收入、總支出、結餘。  
• 右鍵快捷編輯：整合快捷操作邏輯，點選表格帳目後按右鍵，即可直接選擇「編輯」或「刪除」該筆資料。  
• 資料匯出：支援一鍵匯出為 CSV 檔案，方便使用 Excel 等工具進行二次整理。
